Sobre Nós

Este projeto é um exemplo simples de um servidor web feito com Node.js. Ele lida com rotas básicas e arquivos estáticos, servindo conteúdo dinâmico de maneira eficiente.

Nós focamos em criar uma estrutura modular e fácil de entender, ideal para quem está começando a aprender Node.js e quer entender os fundamentos de um servidor web.
